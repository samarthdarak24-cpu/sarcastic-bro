## 🚀 Migration Guide

### From Llama 2 to Gemma 4

```bash
# Step 1: Pull Gemma 4
ollama pull gemma4

# Step 2: Update .env
OLLAMA_MODEL=gemma4

# Step 3: Restart backend
npm run dev

# That's it! No code changes needed.
```

### From Mistral to Gemma 4

```bash
# Step 1: Pull Gemma 4
ollama pull gemma4

# Step 2: Update .env
OLLAMA_MODEL=gemma4

# Step 3: Restart backend
npm run dev
```

### From OpenAI to Gemma 4

```bash
# Step 1: Pull Gemma 4
ollama pull gemma4

# Step 2: Update .env
OLLAMA_URL=http://localhost:11434
OLLAMA_MODEL=gemma4
# Remove or comment out OPENAI_API_KEY

# Step 3: Restart backend
npm run dev

# Benefit: Save ~$30/month!
```
