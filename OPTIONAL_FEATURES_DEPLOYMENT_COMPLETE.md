## 🚀 Deployment Instructions

### Prerequisites
```bash
# Node.js 18+
node --version

# npm 9+
npm --version

# PostgreSQL running
# Redis running
# MQTT broker running (optional)
```

### Installation
```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env
nano .env  # Update with your values

# Run migrations
npm run migrate

# Seed database (optional)
npm run seed
```

### Start Services
```bash
# All services
npm run dev

# Individual services
npm run dev:web    # Frontend
npm run dev:api    # Backend
npm run dev:ai     # AI Service
```

### Verify Deployment
```bash
# Check frontend
curl http://localhost:3000/optional-features

# Check backend
curl http://localhost:3001/api/health

# Check services
npm run test:services
```
