## 🎤 Voice Input - How to Use

```
1. Click the red mic button (🎤)
2. Speak your question clearly
3. See interim text appear in real-time
4. Recording timer shows: 0:15
5. Click mic button again to stop
6. Text auto-fills in input box
7. Click send button
```

**Example:**
- Speak: "What is the best fertilizer for wheat?"
- Text appears: "What is the best fertilizer for wheat?"
- Send message

---

## 📎 File Upload - How to Use

```
1. Click the attachment button (📎)
2. Select image or file from your device
3. File preview appears below input
4. You can add multiple files
5. Click X to remove any file
6. Type your message (optional)
7. Click send button
```

**Example:**
- Select: crop-image.jpg
- Preview shows: 🖼️ crop-image.jpg [X]
- Send message with image

---

## 🔊 Voice Output - How to Use

```
1. Send a message
2. AI responds
3. Response auto-speaks (if AI Voice is ON)
4. See "Speak" button on AI message
5. Click "Speak" to play response
6. Click "Stop" to stop speaking
7. Change language from dropdown
```

**Example:**
- Send: "What is the best fertilizer?"
- AI responds: "The best fertilizer depends on..."
- Response auto-speaks
- Click "Stop" to stop

## 🐛 Troubleshooting

### Mic Not Working?
- Check browser permissions
- Try refreshing page
- Try different browser
- Check microphone is connected

### File Upload Not Working?
- Check file size (max 50MB)
- Check file type (Images, PDF, Office docs)
- Try different file
- Check browser console for errors

### Voice Not Playing?
- Check browser volume
- Check speaker is connected
- Try different browser
- Check browser console for errors

### Language Not Recognized?
- Try English first
- Check language code is correct
- Try different language
- Check microphone quality
