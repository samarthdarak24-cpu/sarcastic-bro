## 🎨 Color System

### Category Colors

```
🟣 AI Intelligence
   Gradient: Purple (#a855f7) → Indigo (#6366f1)
   Use: AI-powered features, insights, intelligence

🟢 Production & Supply
   Gradient: Green (#22c55e) → Emerald (#10b981)
   Use: Products, inventory, crops, quality

🔵 Orders & Logistics
   Gradient: Blue (#3b82f6) → Cyan (#06b6d4)
   Use: Orders, deliveries, tracking, logistics

🟠 Payments & Finance
   Gradient: Amber (#f59e0b) → Orange (#f97316)
   Use: Money, payments, revenue, finance

🌸 Tenders & Bidding
   Gradient: Rose (#f43f5e) → Pink (#ec4899)
   Use: Tenders, bids, auctions, bulk orders

🟡 Trust & Reputation
   Gradient: Yellow (#eab308) → Amber (#f59e0b)
   Use: Reputation, reviews, trust, ratings

🔴 Security & Compliance
   Gradient: Red (#ef4444) → Rose (#f43f5e)
   Use: Security, compliance, audit, protection
```

## 🔄 Navigation Flow

```
Dashboard (Main View)
    │
    ├─→ Click Category Card
    │   └─→ Expands to show features
    │
    ├─→ Click Feature Button
    │   └─→ Opens Feature Page
    │       │
    │       ├─→ View Feature Content
    │       │
    │       └─→ Click "Back to Dashboard"
    │           └─→ Returns to Dashboard
    │
    └─→ Click "Open AgriChat"
        └─→ Opens Chat Interface
```
